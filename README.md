### 🚀 Featured Projects

#### 🏦 [VirtuBank - Digital Banking Architecture](https://github.com/mishrashaksham/VirtuBank)
A dual-architecture banking simulation showcasing the evolution from a strict **C-based DSA engine** to a **Serverless AI-powered SPA**.
- **v1.x (Core C Engine):** Built with dynamic BSTs for `O(log n)` retrieval, array-shifting for transaction logs, and strict memory management guaranteeing zero memory leaks.
- **v2.x (Bharat 2.0 Web SPA):** A serverless architecture using **Firebase Firestore** and **Google Gemini 1.5 Flash**. Features vernacular voice transactions via Web Speech API, algorithmic wealth management, and hyper-localization across 14 languages.

#### 🗳️ [VirtuElect - Next-Gen EVM & AI Platform](https://github.com/mishrashaksham/VirtuElect)
A high-fidelity digital platform simulating the Indian electoral process with AI-driven insights for voters.
- **Tech Stack:** React.js, Tailwind CSS, Node.js, Express.js, Google Gemini API.
- **Features:** Tactile EVM simulator with physical VVPAT generation, "Gyani" (In-memory Vector RAG AI for questioning political manifestos), and a premium dark UI with Neumorphic shadows.

#### 🏛️ [VIT-AP University Website](https://github.com/mishrashaksham/VITAP)
A premium, multi-page interactive website concept for VIT-AP University.
- **Tech Stack:** Vanilla HTML/CSS/JS frontend with a Node.js/Express.js backend.
- **Features:** 6 responsive pages, glassmorphism UI, scroll animations, and 5 RESTful API endpoints backed by lightweight JSON file storage.

#### 💳 [Forage Midas (JPMC Job Simulation)](https://github.com/mishrashaksham/forage-midas)
